Concept(Project Idea) что главное в приложении
Дать общее представление об основных объектах Солнечной системы, их движении, внешнем виде и предоставть краткие сведения.
 -To create a model of the solar system in augmented reality. The
application can be used to obtain visual representation and information about the
objects of the solar system.

User experience
После открытия приложения, когда находятся плоскости, на экране по центру появляется голубой маркер, Который показывает, что есть возможность расположить модель СС.
After opening the application blue marker appiers 

После касания  экрана, появляется полная модель СС, которая вращается.
Когда пользователь касается объекта, то этот объект появляется в большем размере с дополнительной информацией. При повторном касании (или при нажатии на кнопку Назад) возвращается модель СС. Так же можно управлять вращением модели СС и отображением названий объектов.
Меню
-Выход из программы
-Остановить/запустить вращение
-Убрать/добавить названия планет
-Кнопка Назад возвращает из режима просмотра одного объекта к просмотру модели солнечной системы
Content creation
Была использована платформа Unity с компонентами ARCore XR Plugin, AR Foundation.
Все 3D Объекты были созданы с помощью примитивных форм Сфера. Текстуры взяты с сайта  



https://www.solarsystemscope.com/textures/, который предоставляет их в свободное использование. 
Орбиты созданы программно с помощью компоненты GL
Кольца сатурна созданы с помощью скрипта, который задает параметры triangles, vertices, uv у класса Mesh();


Effects
Эффект свечения солнца и пояс астеройдов созданы с помощью Системы частиц.
Эффект атмосферы Земли создан с помощью компоненты Halo.
